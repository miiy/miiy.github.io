# 查看 FPM 状态页

## 配置 FPM

www.conf
```
pm.status_path = /fpm-status
ping.path = /fpm-ping
```

## 配置 Nginx

nginx.conf

```
        location ~ ^/(fpm-status|fpm-ping)$ {
           fastcgi_pass   127.0.0.1:9000;
           fastcgi_param  SCRIPT_FILENAME $fastcgi_script_name;
           include        fastcgi_params;
        }
```

## 重启 PHP-FPM 和 Nginx

```bash
systemctl restart php-fpm
nginx -t
nginx -s reload
```
